# Enhancement prompt library (Smart Modes)

## Summary
Multiple enhancement prompts with icons, descriptions, and quick selection.

## Gap vs Whisper++
SmartFix supports a single prompt; no prompt library or quick switching.

## User Stories
- As a user, I want different enhancement styles (email, summary, coding).
- As a user, I want to reorder and pin my favorite prompts.

## Acceptance Criteria
- Create/edit/delete prompts with title, icon, and template text.
- Select active prompt from keyboard UI.
- Persist prompts in local storage and sync with settings export.
