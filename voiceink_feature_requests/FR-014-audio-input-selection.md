# Audio input selection and priority

## Summary
Allow choosing microphone source and prioritizing external devices when available.

## Gap vs Whisper++
No UI for selecting or prioritizing input devices.

## User Stories
- As a user, I want to use my external mic by default.
- As a user, I want to fall back to the built-in mic if needed.

## Acceptance Criteria
- List available input devices with test capability.
- Priority order and fallback behavior.
- Clear indication of the active input device.
