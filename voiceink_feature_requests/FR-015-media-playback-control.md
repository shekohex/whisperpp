# Media playback control during recording

## Summary
Pause media playback when recording starts and optionally resume after.

## Gap vs Whisper++
No integration with media playback control.

## User Stories
- As a user, I want my music paused automatically while recording.

## Acceptance Criteria
- Toggle for auto-pause/resume media during recording.
- Works with standard Audio Focus and MediaSession APIs.
- Respects other audio focus rules.
