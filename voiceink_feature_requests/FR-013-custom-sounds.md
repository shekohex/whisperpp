# Custom start/stop sounds

## Summary
Let users pick custom audio cues for recording start/stop and test them.

## Gap vs Whisper++
Whisper++ has fixed feedback sounds; no customization.

## User Stories
- As a user, I want distinct sounds that match my preferences.

## Acceptance Criteria
- Select custom audio files for start and stop cues.
- Validate file formats and allow reset to defaults.
- Test playback from settings.
