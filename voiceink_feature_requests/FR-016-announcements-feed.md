# In-app announcements feed

## Summary
Show lightweight announcements (release notes, maintenance notices) inside the app.

## Gap vs Whisper++
No announcements or news feed.

## User Stories
- As a user, I want to see new features and important notices.

## Acceptance Criteria
- Toggle to enable/disable announcements.
- Fetch from a hosted JSON feed with caching.
- Non-intrusive UI with badge/indicator.
