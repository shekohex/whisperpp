# Performance analytics dashboard

## Summary
Analyze transcription performance across models and providers (speed, error rate, usage).

## Gap vs Whisper++
No analytics or performance view exists.

## User Stories
- As a user, I want to compare model speed vs accuracy.
- As a developer, I want data to tune defaults.

## Acceptance Criteria
- Aggregated stats from history (RTF, avg duration, counts).
- Per-model and per-provider breakdowns.
- Exportable summary for diagnostics.
