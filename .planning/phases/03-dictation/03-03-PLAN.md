  <files>

android/app/src/main/java/com/github/shekohex/whisperpp/dictation/DictationController.kt

  <action>
- Add `DictationUndoEntry` containing (minimum):
  - `focusKey`
  - `insertedText`
  - `selectionBeforeInsert` and `selectionAfterInsert` (start/end)

- In `DictationController`:
  - Maintain an undo stack (LIFO) keyed/scoped to the current `FocusKey`.
  - Push exactly one entry per successful send/finalize.
  - Implement undo as conservative safe-apply:
    - Require current focus matches entry focus.
    - Attempt to re-select the previously inserted region and validate the selected text matches `insertedText`.
    - Only then delete and restore cursor to `selectionBeforeInsert`.
    - If any step fails or mismatch: do nothing and surface a short toast ("Can't undo here").

- Undo should remain available even after manual typing when safe-apply passes; otherwise fail safely.
  </action>
  <verify>./android/gradlew -p android :app:assembleDebug</verify>
  <done>
- Multi-step undo works for repeated dictation sends within the same focus.
- Undo never deletes unrelated user text when focus/selection mismatches.
  </done>
</task>
